Установка (Яндекс.Браузер / Chromium)

1) Открой yandex://extensions (или browser://extensions)
2) Включи "Режим разработчика"
3) Нажми "Загрузить распакованное расширение"
4) Выбери папку:
   c:\Users\otloo\project\FileTrace\chrome_extension_yandex

Как пользоваться

1) Нажми на иконку расширения
2) Адрес FileTrace задан в коде расширения (popup.js, константа BASE_URL). При необходимости измени там.
3) Нажми "Открыть" и залогинься на сайте FileTrace (это важно, расширение использует cookies)
4) Нажми "Подключить" (расширение найдёт/откроет вкладку FileTrace и внедрит content.js)
5) Вставь URL -> "Проверить" или "Скачать+анализ"
6) Для файла: выбери .exe -> "Отправить файл"

Отладка

- Кнопка "Ping" показывает URL текущей вкладки FileTrace, с которой работает расширение.
- Ошибки отображаются внизу в статусе.
- При необходимости смотри логи расширения в инструментах разработчика расширений.
